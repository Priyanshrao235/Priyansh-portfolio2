🌐 Personal Portfolio Website

This is a personal portfolio website built using HTML, CSS, and JavaScript to showcase my skills and projects.

🚀 Features

Clean and responsive design
Smooth scrolling navigation
About section with personal introduction
Projects section to showcase work
Contact form with basic interaction
Simple and modern UI
🛠️ Technologies Used

HTML – Structure of the website
CSS – Styling and layout
JavaScript – Interactivity and smooth scrolling
📁 Project Structure

portfolio/
│── index.html     # Main HTML file
│── style.css      # Styling file
│── script.js      # JavaScript functionality
💡 How to Run

Download or clone this repository
Open the folder
Double click on index.html
The website will open in your browser
